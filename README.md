### Installation

_Follow the instructions below to download and install the application._

1. Clone the repo
   ```sh
   git clone https://github.com/enverkoseler/LoginProject.git
   ```
2. Install node_modules
   ```sh
   yarn install
   ```
4. Run project `for ios`
   ```sh
   npx react-native run-ios
   ```
